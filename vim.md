## 编辑操作
| 命令         | 描述                       | 命令 | 描述                     |
|--------------|----------------------------|------|--------------------------|
| a            | 在光标位置后插入           | A    | 在光标所在行尾插入       |
| i            | 在光标位置插入             | I    | 在光标所在行头插入       |
| o            | 在光标所在行下方新增一行   | O    | 在光标所在行上方新增一行 |
| r            | 替换光标位置处一个字符     | R    | 从光标位置开始替换       |
| x            | 删除光标位置处一个字符     | X    | 删除光标位置前一个字符   |
| u            | 撤销上一次的修改           | U    | 撤销当前行上的所有修改   |
| * Ctrl + R * | 恢复上一次撤销的修改       |      |                          |
| J            | 删除光标所在行末尾的换行符 |      |                          |
| ZZ           | 保存文件并退出             |      |                          |
| :qw          | 保存文件并退出             |      |                          |
| :q!          | 强制退出(不保存)           |      |                          |
| :e!          | 强制重新加载原文件(不保存) |      |                          |
| :help        | 使用帮助                   |      |                          |
